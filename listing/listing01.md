Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Вывод: срез [77 78 79]
Так происходит потому, что срез берет индексы с i по j-1 из массива под капотом(наш массив a).
В нашем случае с a[1](77) по a[3](79)

```
